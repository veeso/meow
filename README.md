## Setup

Try running some of the following tasks:

```shell
npx hardhat help
npx hardat compile
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
# deploy
npx hardhat run scripts/deploy.ts
# deploy test net
npx hardhat run scripts/deploy.ts --network mumbai
```
